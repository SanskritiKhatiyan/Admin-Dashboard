This project is a admin dashboard app.

Features included:

1. Product list will be shown with pagination.
2. You can add new product and it will instantly shown in the list.
3. You can edit existing product in the list.
4. You need to be authenticated user to access all above features.
5. You can logout whenever you want.
6. Collapsible sidebar

Tech included:

1. React with Typescript
2. Vite build tool
3. Redux for state management
4. React router dom for routing
5. Ant Design for pre built components
6. Implementing authentication using redux and storing a flag in localstorage with hardcoded values

Project run Setup:

1. Install dependencies
2. Run command: npm run dev

Routes to look after:

1. /login - for authentication
2. /product-list - for list of products
3. /manage-prodcuts - for add and edit products
